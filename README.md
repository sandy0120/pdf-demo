# pdf.js Demo 

## Project introduce
Vite 2.x + Vue 3.x  
  
ThumbMenu -> 縮圖導引 + Infinite Scroll  
PDFRender -> PDF + 放大縮小  
  
Demo Link : https://sandy0120.github.io/pdf-demo/pdf  

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Preview
```
yarn serve 
```
